# VEOS-System-Testing

This repository contains system-level testing artifacts for **VEOS (Vehicle Embedded Operating System)**,
a multi-node automotive platform integrating embedded hardware, Yocto Linux, Android Automotive OS (AAOS),
and a mobile application.

## Project Overview
VEOS connects vehicle hardware, real-time communication, and Android Automotive OS into a unified in-car
and remote monitoring platform.

## Testing Scope
- System testing
- Integration testing
- Manual functional testing
- API & communication validation
- Automotive IVI testing

## System Under Test
- STM32 Bare-metal Car Control Node
- Yocto Middleware Nodes
- Android Automotive OS (AAOS)
- VEOS Mobile Application
- Cloud & networking layer

## Deliverables
- Manual test cases
- Integration test cases
- API / communication test cases
- Bug reports
- Test strategy and execution reports
